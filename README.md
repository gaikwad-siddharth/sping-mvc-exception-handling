# Exception Handling

A Spring Boot web application demonstrating student enquiry form handling using Spring MVC, Spring Data JPA, and layered architecture.

---

## 📌 Project Overview

This project is a Student Enquiry Form application built using Spring MVC.

The application allows users to submit enquiry details such as:
- Student Name
- Email
- Gender
- Course Selection
- Preferred Timings

Submitted data is processed through controller, service, repository, and entity layers following a clean MVC architecture.

---

## 🧠 Concepts Covered

- Spring Boot
- Spring MVC
- Spring Data JPA
- Form Handling
- Data Binding
- Layered Architecture
- MVC Design Pattern
- Database Integration
